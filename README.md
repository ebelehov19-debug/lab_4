## Ползовательские коллекции 

### BookCollection (Списковая коллекция)
Поддерживает методы:
- __len__ — возвращает количество книг.
- __iter__ — позволяет итерироваться по книгам.
- __getitem__ — доступ по индексу и срезам.
- __contains__ - проверяем наличие книги 
- add(book) — добавление книги.
- delite(book) — удаление книги.

### IndexDict (Словарная коллекция)
Индексация по:
- ISBN.
- Автору .
- Году издания .
Поддерживает методы :
- __len__ — количество уникальных книг по ISBN.
- __iter__ — итерация по книгам.
- __getitem__ — доступ по ключу (ISBN, автор, год).
- update_index(book) — обновление индексов при добавлении книги.
- remove_book(book) — удаление книги из всех индексов.
- Методы поиска: find_by_author, find_by_year, find_by_isbn,find_by_title.

## Book (Базовый класс)
Поля: title, author, year, genre, isbn.
Доп методы:
- __repr__ — строковое представление.
- __eq__ — сравнение по ISBN.

## Производные классы 

### Jornal
Добавляет поля: pages, period, publesher.
Методы:
- full_info() — полная информация о журнале.
- size() — вывод количества страниц.
- change_publesher() — смена издательства.
- cnt_jornal_yers() — расчёт количества выпусков в год.

### PriceBook
Добавляет поля: price, quantity, total_sold, total_revenue, is_for_sale.
Методы:
- sell(cnt) — продажа указанного количества книг.
- zakup(quantity, new_price) — закупка книг.
- change_price(newprice) — изменение цены.
- get_sales_statistics() — статистика продаж.

- происходит вызов функции run_simulation() и передаем аргументы step и seed
