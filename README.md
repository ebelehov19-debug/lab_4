# Лабораторная работа №4. Симуляция с пользовательскими коллекциями и псевдослучайной моделью

## Цель работы
- Разработать пользовательский коллекции 
- Реализация псевдослучайной симуляции событий с возможностью воспроизводимости (seed)

## Ползовательские коллекции 

### BookCollection (Списковая коллекция)
Поддерживает методы:
- __len__ — возвращает количество книг.
- __iter__ — позволяет итерироваться по книгам.
- __getitem__ — доступ по индексу и срезам.
- add(book) — добавление книги.
- delite(book) — удаление книги.
